# Tinder-dos-Filmes
O Tinder dos Filmes é uma aplicação web colaborativa e em tempo real que ajuda grupos de pessoas a escolherem um filme para assistir juntos.

# Como Funciona
Você Define os filtros (streaming e gênero), cria uma sala, convida seus amigos, acessam a lobby e todos votam nos filmes que aparecem — quando todo mundo curte o mesmo filme ao mesmo tempo, é MATCH!
